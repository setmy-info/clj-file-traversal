# clj-file-traversal

## Development

Run

```shell
lein run ./test/resources/root
```

Test

```shell
lein test
```

Build documentation

```shell
lein codox
```

Build

```shell
lein uberjar
```

Deploy to Clojars repository

```shell
lein deploy clojars
```
